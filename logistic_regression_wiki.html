Logistic Regression

by Julie PRATX



Definition

Logistic regression is a statistical model for studying the relationships between a set of qualitative variables Xi and a qualitative variable Y. It’s a generalized linear model using a logistic function as a function.

A logistic regression model can also predict the probability of an event happening (value of 1) or not (value of 0) from the optimization of the regression coefficients. This result always varies between 0 and 1. When the predicted value is greater than a threshold, the event is likely to occur, while when this value is below the same threshold, it isn't [1].


Application


However, this method can be adapted to other cases and in particular that of predicting the number of deaths due to SARS-CoV-2 or, more commonly, COVID-19. The aim will be to find out how many deaths can we expect in this pandemic [2].
